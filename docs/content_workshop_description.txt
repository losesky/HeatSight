# ContentWorkshop 界面视觉说明

## 初始界面
初始界面展示了一个两列布局：
- 左侧窄列包含话题搜索和热门话题推荐
- 右侧宽列显示提示信息，引导用户选择话题开始创作
- 整体使用白色卡片、蓝色主题色，风格简洁现代

## 话题选择后界面
用户选择话题后：
- 左侧选中的话题会高亮显示
- 右侧上方卡片展示所选话题的详细信息，包括标题、分类、热度和子话题标签
- 右上角有"生成内容"按钮
- 界面通过蓝色边框和背景色变化清晰指示当前选中的话题

## 内容生成后界面
点击"生成内容"按钮后：
- 右侧原有卡片下方会新增一个卡片，展示生成的内容建议
- 生成内容包括标题建议、内容大纲、关键要点和开篇建议四个部分
- 每个部分都有浅灰色背景区域，内含生成的内容文本
- 每个部分下方有"复制"按钮，方便用户快速复制所需内容
- 卡片顶部有复制全部和下载按钮
